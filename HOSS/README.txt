We provide here the Cubit journal files and list of material / simulation parameters that were used to produce the HOSS simulation results of “Listening for the landing: Estimating the Seismic Detectability of the Mars 2020 EDL Sequence by InSight”. 
The exact content is : 
- A Cubit journal files for creating a 30° slice of an impact target cylinder.
- A Cubit journal file for creating a ‘quarter of sphere’, 180° impact target.
and their associated impactor and mesh ; Parameters used to generate material input files for: 
- A Tillotson equation of state for Tungsten.
- A Lundborg strength equation for Tungsten.
- A Equation Of State for a 44% porosity sand.   
- A Lundborg strength equation for a 44% porosity sand.
